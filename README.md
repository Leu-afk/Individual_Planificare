Comenzi git
git init
Inițializează un repository Git local în folderul curent.
bash
git config --global user.name "Leu-afk"
git config --global user.email "leu478820@gmail.com"
Setează numele și emailul pentru commit-uri (o singură dată pe calculator).
git add .
Adaugă toate fișierele din folderul curent în zona de staging.
git add St_Ind_Proiectare_RO_Leu
Adaugă un fișier specific în staging.
git commit -m "Add SRS document link to README"
Salvează modificările într-un commit cu un mesaj clar.
git remote add origin https://github.com/username/nume-repository.git
Leagă repository-ul local de cel de pe GitHub.
git remote -v
Verifică adresele repository-ului remote.
git push -u origin main
Trimite commit-urile locale pe GitHub, în branch-ul main.
git push
Trimite modificările fără a seta branch-ul (dacă e deja configurat).
git pull
Actualizează repository-ul local cu modificările de pe GitHub.
git status
Afișează starea fișierelor (modificate, adăugate, necomitate).
git log
Afișează istoricul commit-urilor.
git rm nume_fisier.ext
Șterge un fișier din repository și din sistemul de fișiere.
git reset --hard
Revine la ultima versiune salvată, ștergând modificările locale.
